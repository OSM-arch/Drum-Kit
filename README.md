<h1 align="center">Drum Kit 🥁</h1>

<p align="center">Interactive browser drum kit that responds to keyboard and mouse</p>

<h2>🚀 Features</h2>
<ul>
  <li>Play drum sounds with keyboard keys (A-S-D-F-G-H-J-K-L)</li>
  <li>Clickable drum pads with visual feedback</li>
  <li>Responsive design</li>
</ul>

<h2>🎓 Key Learnings</h2>
<table>
  <tr>
    <th>Concept</th>
    <th>Implementation</th>
  </tr>
  <tr>
    <td>DOM Manipulation</td>
    <td><code>querySelector</code>, <code>addEventListener</code></td>
  </tr>
  <tr>
    <td>Audio API</td>
    <td>JavaScript <code>Audio</code> objects</td>
  </tr>
  <tr>
    <td>CSS Animations</td>
    <td>Press effects with <code>transform: scale()</code></td>
  </tr>
  <tr>
    <td>Event Handling</td>
    <td>Keyboard (<code>keydown</code>) + mouse (<code>click</code>)</td>
  </tr>
</table>
